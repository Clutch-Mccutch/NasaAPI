# NasaAPI - Picture of the day
This is an API to grab Nasa's picture of the day. In order to use this script you will need to replace the APIKEY by downloading your own key from Nasa's website here: https://api.nasa.gov/?ref=its-foss. This is the line from the script that you need to add your key: https.get(`${API_URL}?api_key=YOURKEYGOESHERE`.
